Getting Started

Install and secure your first sheet in 60 seconds.

1. Install the Add-on

  1. Open any Google Sheet
  2. Go to Extensions → Add-ons → Get add-ons
  3. Search: Sheet Protector Pro
  4. Click Install

2. Open the Sidebar

Extensions → Sheet Protector Pro → Open Sidebar

3. Protect Your First Range

  1. Select cells (e.g., B2:B10)
  2. Click Refresh in the sidebar
  3. Choose mode: Only You
  4. Click Protect Range
Pro Tip: Use Lock Formulas to auto-protect all formulas.
