Protect Sheet

Lock an entire sheet to prevent accidental changes.

Steps

  1. Select the sheet from the dropdown
  2. Choose protection mode:
    • Warning Only – Warns before edit
    • Only You – Only owner can edit
    • Custom Editors – Add allowed users
  3. Click Protect This Sheet

Unprotect Sheet

Select the sheet → Click Unprotect This Sheet

Live Status

A Locked badge appears next to protected sheets in the dropdown.
