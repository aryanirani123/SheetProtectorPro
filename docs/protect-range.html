Protect Range

Lock specific cells or ranges with fine-grained control.

Steps

  1. Select your range (e.g., C2:C10)
  2. Click Refresh in the sidebar
  3. Enter a description (optional)
  4. Choose protection mode:
    • Warning Only – Shows warning before edit
    • Only You – Only the owner can edit
    • Custom Editors – Add specific emails
  5. Click Protect Range

Edit or Remove

In the Protected Items list:

  • Click the item → Edit panel opens
  • Update description, mode, or editors
  • Click Save or Remove
